# GLA 5: HTTP Requests

This repo contains my GLA 5 assignment using JSONPlaceholder API.

## Implemented in code
- GET all users from `https://jsonplaceholder.typicode.com/users/`
- Display `name`, `email`, `city`, and `company name` in user cards
- Refresh users with a button to re-run GET requests
- DELETE a user by ID using template literals and status checks
- Show request status and errors directly on the page

## Postman tasks
Use Postman for:
- POST create user
- PUT update user id 5

Include screenshots in submission showing method, endpoint, request body, response body, status code, time, and size.

## How to run
1. Open `index.html` in a browser.
2. Click **Refresh Users (GET)** to load user cards.
3. Enter a user ID and click **Delete User (DELETE)**.
